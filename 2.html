Веб-разработка

Веб-разработка - это создание сайтов.

HTML

HTML делает структуру сайта.

В HTML есть теги для заголовков, текста и ссылок.

Без HTML нет веб-страницы.

CSS

CSS делает сайт красивым.

CSS меняет цвета, шрифты и расположение элементов.

С CSS сайт выглядит привлекательно.

JavaScript

JavaScript делает сайт умным.

JavaScript добавляет кнопки, анимацию и реакции на действия.

С JavaScript сайт становится интерактивным.
