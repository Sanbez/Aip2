Задание 4: Объяснение выбора класса colored

Класс назван "colored", а не "green", потому что:

  • Это семантически правильнее - класс описывает состояние, а не конкретный цвет
  • Легко изменить цвет в CSS без изменения JavaScript кода
  • Класс можно переиспользовать для других элементов с разными цветами
